# Comic Creator

An OpenAI Build Week project for helping children turn stories they direct into
finished comics.

The project is currently in product definition. The implementation stack and
MVP scope have not been selected yet.

## Project context

This repository uses a small set of canonical documents so that people and
agents can resume work without reconstructing prior conversations:

- [`AGENTS.md`](AGENTS.md) — operating contract and reading order
- [`docs/PROJECT.md`](docs/PROJECT.md) — stable product brief
- [`docs/RULES.md`](docs/RULES.md) — project and hackathon constraints
- [`docs/STATE.md`](docs/STATE.md) — short snapshot of current reality
- [`docs/PLAN.md`](docs/PLAN.md) — the single active plan
- [`docs/DECISIONS.md`](docs/DECISIONS.md) — append-only durable decisions

For a quick return, read `docs/STATE.md` and then `docs/PLAN.md`.
